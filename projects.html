AI 학습 노트 자동 생성기 Gemini

PDF 기반의 강의 교재를 단 한 번의 클릭으로 깔끔하게 분석하고 노션(Notion)에 자동으로 저장해주는 지능형 웹 애플리케이션입니다. Google Gemini 2.5 Flash API를 활용해 커리큘럼을 생성하고, 상세 실전 마크다운 노트를 작성하는 것이 특징입니다.

✨ Key Features

  • PDF.js를 이용한 브라우저 자체 Text Extraction (Drag & Drop)
  • Gemini 2.5 기반의 스마트 목차 파싱 및 실전 코드/가이드 생성
  • Notion API 연동 (CORS Proxy): 자동 페이지/하위 페이지 트리 구성
  • 사용자 경험 극대화를 위한 Glassmorphism 기반의 마이크로 인터랙션
