      포트폴리오용 구현 범위와 일정

      실제 지원 포트폴리오에서는 2주 MVP + 1주 고도화 구조가 가장 설득력 있습니다.

      1주차

      • 문제 정의, 사용자 시나리오, 와이어프레임
      • 입력 폼, 결과 보드, 담당자 뷰 IA 설계
      • 샘플 데이터 기준 요약/액션 추출 로직 구현

      2주차

      • 우선순위, 마감일, 결정/리스크 분류 고도화
      • 반응형 UI, 빈 상태, 에러 상태, 샘플 체험 흐름 보강
      • 프로젝트 소개 문구와 기획 포인트 정리

      추가 확장

      • OpenAI 또는 Gemini 연동으로 요약 품질 향상
      • 회의 파일 업로드, 음성 파일 전사, 협업 툴 연동
      • 팀별 저장공간과 히스토리 검색 기능 추가
